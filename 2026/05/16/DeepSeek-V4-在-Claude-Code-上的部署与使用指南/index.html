在 Claude Code 上部署 DeepSeek V4

DeepSeek V4 原生兼容 Anthropic 的 API 协议，部署过程非常简单，无需任何第三方工具。

方法一：直接替换后端（推荐）

这是最简单的方式，只需设置几个环境变量即可。

步骤 1：获取 DeepSeek API Key

前往 DeepSeek 官网 注册账号并创建 API Key。

步骤 3：设置环境变量

方法二：使用 deepclaude 项目

deepclaude 是一个开源项目，支持 DeepSeek V4 Pro、OpenRouter、Fireworks AI 等多种后端，号称比原生 Anthropic 便宜 17 倍。

方法三：使用 MCP 服务器优化

DeepSeek-V4-Claude-Code-MCP 是一个开源的 MCP 服务器，利用 DeepSeek V4 的 1M-token 上下文能力，可以索引整个代码仓库，在仓库探索任务中实现 10-100 倍的 token 缩减。

提供 Windows (.exe) 和 Mac (.dmg) 的一键安装程序，无需手动配置。

使用体验与注意事项

优势

  1. 成本大幅降低：从每月 $200 降至约 $7，降幅达 95%
  2. 超长上下文：1M tokens 可一次性处理大型代码库
  3. 原生兼容：无需第三方桥接工具，Claude Code 的所有功能（Agent 循环、工具调用等）均可正常使用
  4. 思考模式：支持 reasoning_effort 参数调整思考强度

当前限制

限制 说明
❌ 不支持图像 V4 是纯文本模型，无法查看截图、UI 设计图、错误截图
⏱️ 超时设置 复杂任务建议设置 API_TIMEOUT_MS=600000
🔄 旧接口即将停用 deepseek-chat / deepseek-reasoner 将于 2026-07-24 停用

与 Sonnet 4.5 的对比评测

根据社区实际使用反馈，DeepSeek V4-Pro 在以下方面表现突出：

  • 代码生成：质量接近 Opus 4.6
  • 问题分析：由于 1M 上下文，能更好地理解大型项目全貌
  • 工具调用：完全兼容 Claude Code 的工具循环
  • 速度：V4-Flash 响应速度极快，适合日常辅助编码

总结

DeepSeek V4 的发布标志着开源大模型的一个重要里程碑。通过简单配置即可将 Claude Code 的后端替换为 DeepSeek V4，以极低的成本获得接近顶级闭源模型的体验。无论是个人开发者还是小型团队，这都是一个极具吸引力的选择。
