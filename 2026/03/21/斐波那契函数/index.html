斐波那契函数

发表于2026-03-21|更新于2026-03-28
|总字数:309|阅读时长:1分钟|浏览量:

斐波那契函数的python代码：

一.设置变量

  • n1 = 0 （n1：代表斐波那契数列的第 0 项（f(0)=0））

  • n2 = 1 （代表第 1 项（f(1)=1））

  • a = 1 （代表当前正在计算的项，初始值设为 1，即第 1 项的值。）

注意：a 的初值在后续循环中会被覆盖，但对 n=1 的情况，循环不执行，直接输出 a=1，正好对应 f(1)。

二. 设置循环

range(2, n + 1) 生成从 2 到 n 的整数序列（包含 2，不包含 n+1 但步长为 1，所以当 n>=2 时，循环至少执行一次）。

每次循环：

1. a = n1 + n2:

计算当前项的值（即前两项之和）。

2. n1 = n2:

将原 n2 的值赋给 n1，为下一次计算做准备（相当于前一项指针后移）。

3. n2 = a

将刚计算出的当前项值赋给 n2，使之成为下一轮计算的“前一项”。

循环变量 j 仅用于控制循环次数，未在循环体内使用。
